# Top Five

Based on the amazing MERN Template from [Coding House](https://github.com/CodingHouseFI/mern-template)

Includes:

* MongoDB / Mongoose
* Express
* React
* Webpack
* Babel
* Bootstrap

and more!

To start:
```
npm install
npm start (or nodemon)
```

#### Note:

Be sure to change the database name in server/app.js to something you like.
